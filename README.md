# Sistema de Matrícula UE JGF 🎓

Este proyecto ha sido desarrollado como parte de los requisitos académicos para la carrera de **Ingeniería en Informática** en la **UNETI** (Universidad Nacional Experimental de las Telecomunicaciones e Informática).

El objetivo principal es proporcionar una solución digital eficiente para la gestión y control de matrículas escolares en la institución **UE JGF**, aplicando estándares de desarrollo de software y gestión de bases de datos aprendidos durante la formación profesional.

---

## 📋 Descripción del Proyecto
El sistema permite automatizar el proceso de inscripción y registro de estudiantes, optimizando el manejo de la información y reduciendo los tiempos de respuesta administrativos.

### 🚀 Funcionalidades Principales
* **Registro de Estudiantes:** Captura de datos personales y académicos.
* **Gestión de Matrícula:** Control de cupos y secciones.
* **Consultas:** Búsqueda rápida de expedientes de alumnos.
* **Interfaz Amigable:** Diseñada para facilitar el uso al personal administrativo.

## 🛠️ Tecnologías Utilizadas
Este proyecto se ejecuta en un entorno de desarrollo local utilizando:
* **Entorno:** XAMPP (Apache, MySQL)
* **Lenguajes:** PHP / HTML5 / CSS3 / JavaScript
* **Base de Datos:** MySQL / MariaDB

## ⚙️ Instalación y Configuración
Si deseas replicar este proyecto localmente:
1. Clona este repositorio o descarga el ZIP.
2. Mueve la carpeta a `C:\xampp\htdocs\`.
3. Importa la base de datos incluida (archivo `.sql`) mediante **phpMyAdmin**.
4. Configura el archivo de conexión si es necesario.
5. Accede desde tu navegador a `localhost/matricula`.
